About Generator

Created when I wanted something to add geometric shapes as images on my page when I wire frame a site I built this tool, it still needs some work but this will provide you with a good start.

Examples

Add this code before your closing body tag 


var images = document.querySelectorAll('.imgGen');
for (var i = 0; i < images.length; i++) {
	var s = new ImageGenerator(images[i], 100, 100);
}



<img class="imgGen">
add a class to an image tag and query that class 

<img class="imgGen" data-size="200">
add data-size to the tag and it will over ride the image options below.
This example will build a 200x200 image 

<img class="imgGen" data-size="200 250">
this example will build a 200x250 size image
